# Generatore di password CLI

Questo è un programma CLI scritto in python che genera password sicure e casuali, puoi scegliere la lunghezza della password e il programma la creerà usando lettere maiuscole, numeri e simboli speciali.

## Come usare

1. Apri il terminale o l'ambiente di sviluppo Python 
2. Esegui il file 'generatore_password.py'
3. Inserisci il numero di caratteri desiderato per la password
4. Il programma ti mostrerà la password generata.
5. Ti verrà chiesto se vuoi generare un'altra password (`s` per sì, `n` per terminare).

## Funzionalità principali

- Genera password di lunghezza personalizzata
- Usa lettere maiuscole, numeri e simboli speciali
- Controlla che la lunghezza inserita sia maggiore di zero
- Puoi generare più password senza riavviare il programma

## Requisiti

- Python 3.x
- Libreria standard `random` (già interna a Python)
